방향키 리듬 게임 - 실행 및 제출 안내

1. 준비 사항
 - 최신 웹 브라우저 (Chrome, Firefox, Edge 등) 권장
 - 인터넷 연결 권장 (사운드 파일이 온라인 링크로 연결되어 있음)

2. 실행 방법
 - 압축을 해제한 후 index.html 파일을 웹 브라우저에서 실행하세요.
 - 게임 시작 화면에서 '속도 조절' 슬라이더로 속도를 조절할 수 있습니다.
 - '게임 시작' 버튼을 눌러 게임을 시작합니다.
 - 방향키(↑, ↓, ←, →)를 타이밍에 맞게 눌러 점수를 얻으세요.
 - 생명이 3번 다 떨어지거나 제한 시간이 지나면 게임이 종료됩니다.
 - 게임 종료 후 최종 점수와 최고 점수를 확인할 수 있습니다.
 - '재시작' 버튼으로 다시 플레이할 수 있습니다.
